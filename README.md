# Formiranje obrazaca kod debelih robota sa ograničenom memorijom

Seminarski rad za potrebe kursa Geometrijski algoritmi na Matematičkom fakultetu Univerziteta u Beogradu. 
Seminarski rad analizira i prezentuje glavne ideje rada *Pattern formation for fat robots with memory*, dostupnog na sledećem linku: https://www.sciencedirect.com/science/article/pii/S0925772125000276
